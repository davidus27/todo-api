# Todo Manager


## Description
Simple to use API that manages ToDo's.
Includes: 
 - Creating new ToDo lists
 - Adding/Updating/Deleting todo's
 - Authentication and Authorization
 - Basic security measures

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
